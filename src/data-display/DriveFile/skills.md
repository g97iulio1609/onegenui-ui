# DriveFile

Use this component to display a Google Drive file or folder as a card.

## Usage

Use this component when:

- The user asks to list files from Google Drive
- Displaying search results from Drive

## Props

- `name`: File name
- `mimeType`: MIME type
- `thumbnailLink`: URL to thumbnail image
- `webViewLink`: Link to open in Drive
- `owners`: Array of owner objects
